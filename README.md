# TrueForm
<p align="start">    
Simple and functional, discover the True Form's of your fitness movements! </br>
</p>  
</br>  

## App Features (v1)
* 100% Kotlin,
* MVVM with Use Case pattern,
* Single Activity pattern,
* Coroutines,
* Flow,
* Jetpack Compose,
* Jetpack Navigation,
* Dependency Injection with Dagger-Hilt,
* Material Design,
* Coil,
* Timber,
* OkHttp
